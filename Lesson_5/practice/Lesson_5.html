Lesson 5: Object & Arrays in JavaScript

1. Object

  • An object is a collection of key-value pairs.
  • The key is always a string.
  • The value can be any data type (string, number, boolean, array, another object, etc.).
  • An object is created using curly braces {}.
  • Object properties can be accessed using dot notation (object.key) or bracket notation (object['key']).

syntax of Object

let objectName = {
    key1: value1,
    key2: value2,
    // ...
};

2. Example of Object

var person = {
    name: "John",
    age: 30,
    gender: "male",
    isMarried: false,
    children: ["Alice", "Bob"],
    address: {
        street: "123 Main St",
        city: "New York",
        state: "NY",
        zip: "10001"
    }
};

Accessing Object Properties

person.name;      // "John"
person["age"];    // 30

Modifying Object Properties

person.name = "Jane";
person["age"] = 25;

Adding a New Property

person.email = "jane@example.com";
person["phone"] = "123-456-7890";

Deleting a Property

delete person.gender;

3. Real Example Walkthrough

Let's use this object for our examples:

var person = {
    name: "John",
    age: 30,
    gender: "male",
    isMarried: false,
    children: ["Alice", "Bob"],
    address: {
        street: "123 Main St",
        city: "New York",
        state: "NY",
        zip: "10001"
    }
};

Accessing Properties

person.name;            // "John"
person.age;             // 30
person.children[0];     // "Alice"
person.address.city;    // "New York"

Modifying Properties

person.name = "Jane";
person.age = 25;
person.isMarried = true;
person.children[0] = "Emily";
person.address.city = "Los Angeles";
person.address.state = "CA";

Adding New Properties

person.email = "jane@example.com";
person.phone = "123-456-7890";

Deleting Properties

delete person.email;
delete person.phone;

4. Arrays

  • An array stores multiple values in a single variable.
  • It is an ordered collection of values.
  • An array is created using square brackets [].
  • Elements are accessed by their index, starting from 0.

syntax of Array

let arrayName = [
    value1,
    value2,
    // ...
];

5. Example of Array

var fruits = ["apple", "banana", "orange", "grape"];

Accessing Array Elements

fruits[0];   // "apple"
fruits[1];   // "banana"

Modifying Array Elements

fruits[0] = "pear";
fruits[1] = "mango";

Adding a New Element (to the end)

fruits.push("kiwi");

Deleting an Element (from the end)

fruits.pop();

6. Iterating Over Arrays

You can loop through array elements in several ways.

for loop

The traditional way to loop, giving you access to the index i.

var fruits = ["apple", "banana", "orange"];
for (var i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

forEach loop

A modern, cleaner method available on arrays.

var fruits = ["apple", "banana", "orange"];
fruits.forEach(function(fruit) {
    console.log(fruit);
});

for...of loop

Another modern and very readable way to loop over iterable values like arrays.

var fruits = ["apple", "banana", "orange"];
for (var fruit of fruits) {
    console.log(fruit);
}
