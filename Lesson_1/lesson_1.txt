Lesson 1 : Introduction to JavaScript

1. Introduction to JavaScript
    - JavaScript is a programming language that is used to create interactive and dynamic web pages.
    - It is a client-side scripting language, which means it is executed on the user's browser rather than on the server.
    - JavaScript can be used to add interactivity to web pages, such as responding to user input, displaying dynamic content, and creating animations.

2. Why learn JavaScript?
    - It’s the most popular programming language in web development.
    - It’s easy to learn and use.
    - It’s widely supported by all modern web browsers.
    - It’s used in a wide range of applications, from simple web pages to complex web applications.

3. Where is JavaScript used?
    + Frontend (Browser):
        - JavaScript is used to create interactive and dynamic web pages.
        - It can be used to add interactivity to web pages, such as responding to user input, displaying dynamic content, and creating animations.
        - It can also be used to create complex web applications, such as online stores, social media platforms, and more.
    + Backend (Server):
        - JavaScript can also be used on the server-side to create dynamic web applications.
        - It can be used to create APIs, handle requests and responses, and perform other server-side tasks.
    + Other Uses:
        - JavaScript can also be used in other areas, such as mobile app development, game development, and more.

4. How to write JavaScript code?
    + There are 3 way: 
        - Inline JavaScript: This is when you write JavaScript code inside HTML elements using the script tag.
        - Internal JavaScript: This is when you write JavaScript code inside HTML file using the script tag.
        - External JavaScript: This is when you write JavaScript code in a separate file with a .js extension and link it to your HTML file using the script tag.

    + Example:
