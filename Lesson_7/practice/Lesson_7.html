Lesson 7: JavaScript Events & DOM

1. What is DOM?

DOM (Document Object Model) is a tree-like structure that represents your HTML page. JavaScript can use the DOM to:

  • Change text, images, and styles dynamically.
  • Read content from a web page.
  • Add or remove elements from a web page.
  • React to user actions.

Example:

Hello

I LOVE

Code:
const title = document.getElementById("title");
const btn = document.getElementById("btn");
btn.addEventListener("click", function() {
    title.innerText = "Hello World";
});
Note:
 .innerText / .textcontent        // Change text
 .innerHTML                       // Change HTML
 .style.property                  // Change color
 .createElement()                 // add new element
 .appendChild()                   // add new element
 .remove()                        // remove element
 .classList.add()                 // add class
 .classList.remove()              // remove class
 .addEventListener()              // add event listener

2. Selecting Elements in the DOM

You can access HTML elements using JavaScript in several ways:

Modern Methods (Recommended)

// Selects the element with Class
let firstItem = document.querySelector('.list-item');

// Selects the element with ID
let title = document.querySelector('#main-title');

// Selects the element with All matches
let allItems = document.querySelectorAll('.list-item');

Older Methods

// Selects the element with Class
let itemsByClass = document.getElementsByClassName('list-item');

// Selects the element with ID
let titleById = document.getElementById('main-title');

// Selects the element with all tag
let itemsByTag = document.getElementsByTagName('li');

Example

document.querySelector("#idName");     // by ID
document.querySelector(".className");  // by class
document.querySelector("p");           // by tag
document.querySelector("div p");       // nested elements

Combine your knowledge to dynamically add and remove elements.

Note:

const container = document.querySelector('#container');

reverseTextBtn.addEventListener('click', () => {
    const text = document.querySelector('#text');
    if (text) text.textContent = 'Hello, World!';

5. DOM Events (User Interaction)

An event is something that happens in the browser:

  • A user clicks a button
  • A user types in an input
  • The page loads
  • The mouse moves, form submits
Code:
click                   // User clicks on an element
dblclick                // User double clicks on an element
mouseover               // User hovers over an element
mouseout                // User stops hovering over an element
keydown                 // User presses a key on the keyboard
keyup                   // User releases a key on the keyboard
input                   // User types into an input field
submit                  // User submits a form
change                  // User changes the value of an input field
focus                   // User focuses on an input field
load                    // Page has finished loading

6. DOM Traversing (Navigating the DOM Tree)

Traversing means navigating the DOM tree ‚Äî moving from one element to another:

  • From parent ‚Üí child
  • From child ‚Üí parent
  • From one sibling ‚Üí another

Paragraph 2

Example:
<div id="container-element">
    <p class="element-p">Paragraph 1</p>
    <p class="element-p">Paragraph 2</p>
</div>
Note:
.parentElement                    // Move up one level
.children                         // Get all child elements
.firstElementChild                // Get first child element
.lastElementChild                 // Get last child element
.nextElementSibling               // Get next sibling element
.previousElementSibling           // Get previous sibling element
.children[n]                      // Get specific child element
Accessing Parent Elements:
// Code:

// index.html
<div id="container-element">
    <p class="element-p">Paragraph 1</p>
    <p class="element-p">Paragraph 2</p>
</div>

// script.js
const p = document.querySelector('.element-p');
console.log(p.parentElement);
Accessing Child Elements:
// Code:

// index.html
<div id="container-element">
    <p class="element-p">Paragraph 1</p>
    <p class="element-p">Paragraph 2</p>
</div>

// script.js
const containerElement = document.getElementById('container-element');
const elementP = document.getElementsByClassName('element-p');
console.log(containerElement.children); // HTMLCollection
console.log(containerElement.childNodes); // NodeList
console.log(elementP[0].parentNode); // Node
console.log(elementP[0].nextSibling); // Node
console.log(elementP[0].previousSibling); // Node
Example:
<div id="profile">
    <h2>Subject:</h2>
    <ul>
        <li>JavaScript</li>
        <li>PHP</li>
        <li>Node.js</li>
    </ul>
</div>

<script>
    const profile = document.getElementById("profile");

    // Access children
    console.log(profile.children); // h2 + ul

    // Access list items
    const list = profile.lastElementChild; // <ul>
    console.log(list.children); // li elements

    // Access siblings
    const firstSkill = list.firstElementChild;
    const nextSkill = firstSkill.nextElementSibling;
    console.log(nextSkill.innerText); // "PHP"

    // Access parent
    console.log(firstSkill.parentElement); // <ul>
</script>
